# Git-Style Version Control System in C++

## Description
A simplified Git-like version control system built in C++ that supports committing file snapshots, viewing commit history, and restoring previous versions using object-oriented programming and file handling.

## Features
- Initialize repository
- Commit file snapshots
- View commit history (log)
- Checkout previous versions
- File-based version storage

## Concepts Used
- Object-Oriented Programming
- File Handling
- Vectors (STL)
- System Design basics
- Snapshot-based version control

g++ main.cpp Repository.cpp Commit.cpp FileManager.cpp -o git
